Detailed Description

This class serves as a service layer for handling user-related business logic such as registration, profile updates, password recovery, and OTP verification.

It interacts with the UserRepository and PatientRepository to perform database operations related to users. It also implements the UserDetailsService interface to integrate with Spring Security for authentication purposes.
