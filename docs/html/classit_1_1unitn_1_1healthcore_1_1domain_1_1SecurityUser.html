Detailed Description

Adapter class that implements UserDetails to integrate our User entity with Spring Security.

@detail It provides the necessary user information to Spring Security for authentication and authorization purposes. @detail The getAuthorities method maps the user's role to a GrantedAuthority, which Spring Security uses to manage access control.
