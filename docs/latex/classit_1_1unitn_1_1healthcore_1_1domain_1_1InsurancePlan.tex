\doxysubsection{Detailed Description}
Represents an insurance plan in our system, mapped to the \char`\"{}insuranceplans\char`\"{} table in the database. 

It maintains a many-\/to-\/many relationship with \mbox{\hyperlink{classit_1_1unitn_1_1healthcore_1_1domain_1_1Hospital}{Hospital}}, indicating which hospitals accept this insurance plan.
