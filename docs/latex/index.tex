 \hypertarget{index_autotoc_md1}{}\doxysection{Overview}\label{index_autotoc_md1}
Health\+Core is a web-\/based healthcare management system developed for a small network of hospitals. The system aims to provide a secure and efficient platform that connects patients, doctors, and administrators while ensuring the protection of sensitive medical information.

The project was developed as part of a software engineering course (University of Trento) and follows a layered architecture designed to support maintainability, scalability, and security.\hypertarget{index_autotoc_md2}{}\doxysection{Project Structure}\label{index_autotoc_md2}

\item A compatible Java web server (e.\+g. Apache Tomcat)
\item A relational database Postgre\+SQL
\item Maven
\end{DoxyItemize}\hypertarget{index_autotoc_md5}{}\doxysubsection{Setup}\label{index_autotoc_md5}

\begin{DoxyEnumerate}
\item {\bfseries{Clone the repository}} 
\begin{DoxyCode}{0}
\DoxyCodeLine{git clone https://github.com/theThirtyOnePercent/HealthCore.git}
\DoxyCodeLine{cd HealthCore}

\end{DoxyCode}

\item {\bfseries{Configure your database}} ~\newline
 Update the database connection settings in the application configuration file with your local credentials.
\item {\bfseries{Build the project}} 
\begin{DoxyCode}{0}
\DoxyCodeLine{mvn clean install}

\end{DoxyCode}
 $\ast$(Or import into your IDE and build from there.)$\ast$

 Deploy the generated {\ttfamily .war} file to your web server (e.\+g. Tomcat\textquotesingle{}s {\ttfamily webapps/} directory).
\item {\bfseries{Access the application}} ~\newline
 Open your browser and navigate to\+: 
\begin{DoxyCode}{0}
