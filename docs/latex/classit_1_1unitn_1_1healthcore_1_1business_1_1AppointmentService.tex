\mbox{\hyperlink{classit_1_1unitn_1_1healthcore_1_1business_1_1AppointmentService}{Appointment\+Service}} is a service class that provides business logic related to appointments. 

It interacts with the \mbox{\hyperlink{classit_1_1unitn_1_1healthcore_1_1business_1_1UserService}{User\+Service}} to retrieve the current user and the Appointment\+Repository to access appointment data. The service includes methods for retrieving future and past appointments, as well as getting detailed information about specific appointments. @detail The \mbox{\hyperlink{classit_1_1unitn_1_1healthcore_1_1business_1_1AppointmentService}{Appointment\+Service}} class is responsible for handling appointment-\/related operations in the system. It provides methods to retrieve future and past appointments for the currently authenticated user, as well as detailed information about specific appointments.
