\doxysubsection{Detailed Description}
This class serves as a REST controller for handling user-\/related operations such as registration, profile updates, password recovery, and OTP verification. 

This class serves as a base for specific user roles like Patient, Doctor, and Administrator. It uses JPA annotations for ORM mapping to the \char`\"{}users\char`\"{} table in the database. \begin{DoxySeeAlso}{See also}
